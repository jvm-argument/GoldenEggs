# GoldenEggs

Плагин с "загадочными яйцами" для спавнеров. Вставляешь яйцо в спавнер (ПКМ),
и он начинает спавнить рандомных мобов по шансам из конфига. Что показывает
спавнер - то и заспавнит.

Ещё есть Золотая кирка Джейка: ломаешь ей любой спавнер - он выпадает предметом
с сохранённым мобом внутри, а кирка после этого ломается. Если спавнер был с
загадочным яйцом - выпадет особый спавнер (в названии видно какое яйцо), и при
установке обратно привязка восстановится. Обычный спавнер падает с названием по
мобу (сломал зомби-спавнер - выпал спавнер ZOMBIE).

Команды:
/goldenegg give ник яйцо кол-во
/goldenegg pick ник - выдать кирку
/goldenegg reload

Собирается через mvn clean package, нужна java 21 и paper 1.21.
