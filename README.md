# GoldenEggs

Плагин с "загадочными яйцами" для спавнеров. Вставляешь яйцо в спавнер, и он
начинает спавнить рандомных мобов по шансам из конфига.

Пока в разработке, не всё дописано. Базовая структура есть, бд подключается,
конфиг читается, дальше доделываю парсинг яиц, сами события и команды.

Команды думаю сделать так:
/goldenegg give <ник> <яйцо> [кол-во]
/goldenegg reload

Собирается через mvn clean package, нужна java 21 и paper 1.21.
